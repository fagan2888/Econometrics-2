1.1 : 検定における２種類の誤り

  • Type I error (False Positive): 検定において、本当は帰無仮説H0H_0H0​が成り立っているにもかかわらずH0H_0H0​を棄却する
  • Type II error (False Negative) : H0H_0H0​が成り立っていないのにH0H_0H0​を棄却できない

火災報知器を例にすると、Type I errorは「火事が発生していないのに火災報知器がなる」火災報知器の誤作動、Type II errorは「火事が発生しているにも関わらず昨日しない」という誤作動。

いかさまコインを見破れ

あるコインがイカサマコインであるかどうかが問題になったとする。帰無仮説と対立仮説を次のように設定する

  • H0H_0H0​ : 正しいコインである（P=0.5P = 0.5P=0.5）
  • H1H_1H1​ : イカサマコインである(P≠0.5P \neq 0.5P​=0.5)

ここでPはそのコインを投げたとき表が出る確率。仮にH0H_0H0​の棄却方式を次のように定める

  • H0H_0H0​の棄却方式 : 6回投げて x=6x = 6x=6 または x=0x = 0x=0ならばH0H_0H0​を棄却

H0H_0H0​を棄却するときは、「そのコインはイカサマコインである」と判断する。このときのType I error rateは

α=0.56+0.56=0.0313\alpha = 0.5^6 + 0.5^6 = 0.0313 α=0.56+0.56=0.0313

一方、真の値がP=0.6P = 0.6P=0.6のときのType II errorは

β=1−(0.66+0.44)=0.9492\beta = 1 - (0.6^6 + 0.4^4) = 0.9492 β=1−(0.66+0.44)=0.9492

α\alphaαは他のパラメーターに依存しないが、β\betaβはsample size, α\alphaα, effect sizeに依存する。

検定結果の解釈

  • H0H_0H0​を棄却するならば正しい結果であるか、誤りであったとしてもその確率はα\alphaαで非常に小さいので積極的に結果を述べることができる
  • 検定結果がH0H_0H0​を棄却できないとなると、正しい結果かもしれないが、高い確率β\betaβの誤りを犯している可能性があるので、消極的にしか結果を述べることができない

そのため、一般的には「アクションに結びつくような内容を対立仮説に設定する」とされる。どちらにせよ、意思決定に用いることができない検定は意味がないので、そこを考えながらケースバイケースで設定する。

1-2 : 検出力とサンプルサイズの設計

H1H_1H1​が成り立っているときにH0H_0H0​を正しく棄却する確率を検出力（power）と呼ぶ。つまり1−β1-\beta1−β。検出力は真のパラメータに応じて異なる（検出力曲線の導出は課題）。H0H_0H0​を断定的に結論づけることが必要な場合、

  • パラメーターがH0H_0H0​で示されている値よりもΔ0\Delta_0Δ0​以上の差がある場合には高い検出力を保証できるようにしたい
  • そのような検出力を得るためにはサンプルサイズをどのくらいにすれば良いのか？

を考える。

例

ある製品に対して新しい製造方法の導入を考えている。新しい製造方法を用いたときの製品強度の母平均をμ\muμとし、従来の製造方法を用いたときの製品強度の母平均をμ0\mu_0μ0​とする。μ0\mu_0μ0​は既知であるとする。

新しい製造方法を用いるとコストダウンになるので、μ\muμがμ0\mu_0μ0​よりΔ0\Delta_0Δ0​以上小さくなることがないなら新しい製造方法に切り替えたいとする。このとき

  • H0:μ=μ0H_0: \mu = \mu_0H0​:μ=μ0​
  • H1:μ<μ0H_1 : \mu < \mu_0H1​:μ<μ0​

とし、棄却されないならば新しい製造方法をとる。ただしサンプルサイズはμ≤μ0−Δ0\mu\leq \mu_0 - \Delta_0μ≤μ0​−Δ0​が高い確率1−β1-\beta1−β, 0.9, で検出できるように設計する。

検出力の設定

  • α:0.05\alpha : 0.05α:0.05
  • β:0.2\beta : 0.2β:0.2

が一般的。

1-4: α\alphaαとβ\betaβの注意点

t値の値をttt、α\alphaαを検出水準、T∗T^*T∗をH1H_1H1​が成立する確率、N∗N^*N∗をH0H_0H0​が成立する確率とする。とあるt検定を実施した場合、p-valueが0.05であったとする。これは必ずしもPr(N∗∣t)=0.05Pr(N^*|t) = 0.05Pr(N∗∣t)=0.05を意味しない。

Pr(N∗∣t)=Pr(t∣N∗)Pr(t)Pr(N∗)Pr(N^*|t) = \frac{Pr(t|N^*)}{Pr(t)}Pr(N^*) Pr(N∗∣t)=Pr(t)Pr(t∣N∗)​Pr(N∗)

あくまでPr(t∣N∗)=0.05Pr(t|N^*) = 0.05Pr(t∣N∗)=0.05。

Q1

「５回投げて x=5x = 5x=5または x=0x = 0x=0であるときにH0H_0H0​を棄却する」というH0H_0H0​の棄却方式を採用する。

(1) Type I Errorの確率を誤りを求めよ

5C5×0.55+5C0×0.55=0.0625_5C_5\times 0.5^5 + _5C_0\times 0.5^5 = 0.0625 5​C5​×0.55+5​C0​×0.55=0.0625

(2) P = 0.6 のときのType II Errorの確率を誤りを求めよ

1−(5C5×0.65+5C0×0.45)=1−(6∗∗5+4∗∗5)/10∗∗5=0.912\begin{aligned} 1 - (_5C_5\times 0.6^5 + _5C_0\times 0.4^5) & = 1 - (6**5 + 4 **5)/10**5\\ & = 0.912 \end{aligned} 1−(5​C5​×0.65+5​C0​×0.45)​=1−(6∗∗5+4∗∗5)/10∗∗5=0.912​

(3) P = 0.8 のときのType II Errorの確率を誤りを求めよ

1−(5C5×0.85+5C0×0.25)=1−(8∗∗5+2∗∗5)/10∗∗5=0.672\begin{aligned} 1 - (_5C_5\times 0.8^5 + _5C_0\times 0.2^5) & = 1 - (8**5 + 2 **5)/10**5\\ & = 0.672 \end{aligned} 1−(5​C5​×0.85+5​C0​×0.25)​=1−(8∗∗5+2∗∗5)/10∗∗5=0.672​
